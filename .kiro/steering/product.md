# Product Overview

This is a modern portfolio website for Muhamad Ali Ridho, a tech enthusiast. The portfolio showcases projects, skills, and professional experience with a focus on premium UI/UX and performance.

## Key Features

- **Interactive Design**: Custom cursor effects, smooth animations, and micro-interactions
- **Content Management**: Integrated with Contentful CMS for dynamic content
- **Performance Focused**: Built with Astro for optimal loading speed and SEO
- **Accessibility**: WCAG 2.1 AA compliant with proper keyboard navigation
- **Responsive**: Optimized for mobile, tablet, and desktop experiences
- **Theme Support**: Dark/light mode with system preference detection

## Target Audience

- Potential employers and clients
- Fellow developers and tech enthusiasts
- Anyone interested in AI, IoT, web development, and emerging technologies

## Content Types

- **Projects**: Technical projects with categories (web, mobile, IoT, AI/ML)
- **Skills**: Technical expertise and proficiency levels
- **Professional Journey**: Career timeline and experiences
- **Tech Interests**: Areas of focus and emerging technology interests
