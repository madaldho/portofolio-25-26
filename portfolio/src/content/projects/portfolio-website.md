## Overview

My personal portfolio website showcasing my projects, skills, and journey as a Tech Enthusiast. Built with performance and aesthetics in mind.

## Features

- Stunning animations and micro-interactions
- Custom cursor effects
- Dark/Light mode with smooth transitions
- Responsive design for all devices
- SEO optimized
- Fast loading with Astro's static generation

## Technical Details

Built with Astro for optimal performance, styled with TailwindCSS, and enhanced with custom CSS animations. Deployed on Vercel with automatic deployments from GitHub.
