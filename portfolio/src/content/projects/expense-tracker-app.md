## Overview

A beautifully designed expense tracking app that helps users manage their finances with intuitive visualizations and smart insights.

## Features

- Add and categorize expenses
- Monthly budget setting and alerts
- Beautiful charts and statistics
- Cloud sync across devices
- Export to CSV/PDF
- Dark mode support

## Technical Details

Built with Flutter for cross-platform compatibility (iOS & Android). Uses Firebase for authentication and real-time database. State management with Riverpod and FL Chart for data visualization.
