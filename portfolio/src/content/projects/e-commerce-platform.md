## Overview

A modern e-commerce platform with a clean, user-friendly interface and powerful admin capabilities for managing products, orders, and customers.

## Features

- Product catalog with search and filtering
- Shopping cart and checkout process
- Stripe payment integration
- User authentication and profiles
- Admin dashboard for inventory management
- Order tracking and notifications

## Technical Details

Built with Next.js 14 using App Router, Prisma ORM with PostgreSQL database, and Stripe for secure payment processing. The admin dashboard provides comprehensive analytics and management tools.
