## Overview

A comprehensive weather monitoring system that collects environmental data using various sensors and displays real-time information on a web dashboard.

## Features

- Temperature and humidity monitoring
- Atmospheric pressure measurement
- Air quality index tracking
- Historical data visualization
- Mobile-responsive dashboard
- Real-time alerts for extreme conditions

## Technical Details

The system uses ESP32 microcontroller with DHT22 and BMP280 sensors. Data is transmitted via WiFi to Firebase Realtime Database and visualized using Chart.js on a responsive web interface.
