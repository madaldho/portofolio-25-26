## Overview

A sophisticated AI assistant that can answer questions based on uploaded documents using Retrieval-Augmented Generation (RAG).

## Features

- Document upload and processing (PDF, DOCX, TXT)
- Context-aware conversations
- Semantic search across documents
- Conversation history
- Export chat transcripts

## Technical Details

Built with LangChain for orchestration, Pinecone for vector storage, and OpenAI's GPT-4 for generation. The backend is powered by FastAPI with a React frontend for the chat interface.
