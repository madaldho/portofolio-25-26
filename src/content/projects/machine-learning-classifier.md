## Overview

A machine learning project that classifies images into multiple categories using a custom-trained convolutional neural network with a user-friendly web interface.

## Features

- Real-time image classification
- Support for multiple image formats
- Confidence score display
- Batch processing capabilities
- Model performance metrics
- RESTful API for integration

## Technical Details

The model is built using TensorFlow/Keras with a custom CNN architecture. The web interface is powered by FastAPI with real-time prediction capabilities. Deployed using Docker for scalability and consistency.
